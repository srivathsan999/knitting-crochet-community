Nordic Winter Beanie

by Emma Crafts

$3.50

A charming Nordic-inspired beanie featuring traditional Fair Isle colorwork patterns. This intermediate pattern will teach you stranded knitting techniques while creating a stunning winter accessory.

Skill Level Intermediate
Estimated Time 5 Days
Yarn Weight Sport (2)
Needle Size US 5 / 3.75mm
Gauge 24 sts x 28 rows
Language English

Description

This beautiful Nordic beanie features authentic Fair Isle patterns inspired by traditional Scandinavian designs. The pattern includes both charted and written instructions for the colorwork sections.

Learn the art of stranded colorwork with this comprehensive pattern. Includes tips for managing multiple colors, maintaining even tension, and creating neat color changes. The finished hat features a cozy folded brim and optional pompom.
